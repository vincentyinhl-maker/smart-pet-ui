# 智能宠物用品 - UI/UX设计稿描述与指南

**状态**: 初稿
**作者**: 专业 UI/UX 设计师 (UI/UX Designer)

## 1. 整体设计美学与风格 (Aesthetics)
- **主题风格**: 极简、现代、玻璃拟物化 (Glassmorphism)。要给人一种高级智能家居硬件的控制感，科技感中带点宠物的温馨。
- **配色方案 (Color Palette)**:
  - **背景色**: 柔和的深灰色深色模式(如 `#111827`)，或者柔和的米白(`bg-slate-50`)。我们将采用“深色科技风”为主基调，以凸显硬件设备的质感和视频画面的对比。
  - **核心点缀色 (Primary)**: 高级极光绿 (如 Tailwind 的 `teal-400` 或 `emerald-500`)，表示设备在线、健康、成功。
  - **副色/警示色 (Warning)**: 琥珀橙 (`amber-500`)，用于余粮不足警告。
- **字体建议**: `Inter` 或 `Outfit`。强调无衬线字体的几何美感。

## 2. 核心界面布局设计

### 2.1 登录页 (Login Page)
- **背景**: 全屏渐变背景，带有虚化的宠物剪影或设备发光轮廓。
- **交互组件**: 毛玻璃效果 (`backdrop-blur`) 的登录卡片。
- **内容**: Logo标题，邮箱/密码输入框（带微小的focus呼吸灯效果动画），柔和的高亮渐变提交按钮。

### 2.2 设备主页 (Dashboard)
- **顶栏**: 问候语 ("Hello, 猫主子")，右侧用户头像。
- **主要大卡片模块 (Device Card)**:
  - 左侧: 设备简笔插画/图标。
  - 右侧状态区: 设备在线指示灯（心跳动画），余粮百分比进度环（Circular Progressbar）。
- **底栏 (Bottom Navigation)**: 主页 (Home)，记录 (History)，控制 (Control)，设置 (Settings)。带有丝滑切换动画。

### 2.3 远程控制页 (Remote Control)
- **上半部分 (Video Stream)**: 圆角视频播放框区域，左上角标有 "Live", 在没有真实推流时使用宠物互动循环视频占位或优雅骨架屏。
- **下半部分 (Action Panel)**: 
  - 出粮克数滑动条 (Slider)
  - 正中心一个极其吸睛的大圆形“喂食 (Feed)”按钮。点击时要求有明显的物理按压波纹动画 (Ripple) 以及微弱的主题色晕染。
- **喂食记录 (Recent Logs)**: 下半部分有一个抽屉式的滚动面板，罗列最近3次的喂食记录列表。

### 2.4 设置页 (Settings)
- 列表式布局，每行包含图标和标签： "设备管理", "报警通知设定", "关于", "退出登录"。
- 采用简单的右箭头指引设计。

## 3. 动效设计建议 (Micro-animations)
1. **喂食按钮**: 必须提供点击下沉+波纹反馈。
2. **设备上线提示**: 在线灯不仅要绿，还要有缓慢的Opacity呼吸动画，赋予设备“生命力”。
3. **页面切换**: `framer-motion` 或纯 CSS `transition` 实现无缝淡入淡出。
